MIME Type Information

A MIME type (Multipurpose Internet Mail Extensions) is a standard that indicates the nature and format of a document, file, or assortment of bytes. Web servers and browsers use MIME types to determine how to process a URL, so it's important that web servers send the correct MIME type in the response's Content-Type header.

This page helps you test if your web server correctly identifies and serves various file types with their appropriate MIME types.

Supported MIME Types

File Extension MIME Type Description
.txt text/plain Plain text files
.html, .htm text/html HTML documents
.css text/css Cascading Style Sheets
.js text/javascript JavaScript files
.json application/json JSON data interchange format
.xml application/xml XML documents
.png image/png PNG images
.svg image/svg+xml SVG vector images
